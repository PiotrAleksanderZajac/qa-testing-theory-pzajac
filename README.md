# 🧪 Software Testing Theory

A structured, practical knowledge base covering the core theory of software testing.  
Designed as a reference for **QA/SDET interviews**, daily engineering work, and continuous learning.

This repository contains concise explanations, examples, models, heuristics, and test design techniques — all organized into clear modules.

---

## 🎯 Goals of This Repository
- Provide a clean, structured summary of essential testing theory  
- Serve as a preparation resource for QA/SDET interviews  
- Offer practical examples and explanations, not just definitions  
- Organize testing knowledge into modules that can be expanded over time  
- Act as a personal knowledge base for continuous improvement
